1.前置准备

1.首先要安装好Git ,下面是官方网址,大家可以自行去安装

Git

安装完毕，鼠标右键，看到 Git Bash Here 就显示已经安装好了Git ，可以使用相关命令进行操作

关于Git的常见命令，可以参考下面的文章

Git 常用基本命令使用详细大全_git命令行-CSDN博客

GitHub

2.安装node.js

Node.js — 在任何地方运行 JavaScript

输入 node -v 命令

3.安装Hexo

安装命令：npm install hexo -g

这是因为权限问题，可以尝试以下方法：

重新运行命令：再次尝试运行你的npm命令：

前端-学习记录

ES6

简介

ES 全称 EcmaScript ,是脚本语言的规范，而平时经常编写的 JavaScipt ,是EcmaScript 的一种实现

ES新特性其实是指 javaScript 的新特性

ECMAScript是由Ecma国际通过ECMA-262标准化的 脚本程序设计语言

let

注意：1.变量不能重复声明

​ 2.块级作用域，只在作用域中有效，外面则读取不到

​ 3.不存在代码提升：使用原来的 var 可以将未被赋值的变量打印为 undefined,但是 let 则不允许，未被初始化不允许进行打印

​ 4.不影响作用域链

实践案例

const

注意：1.一定要赋初始值

​ 2.（潜规则）一般常量使用大写

​ 3.常量的值无法修改

​ 4.块级作用域，只能在相应的作用域中使用，离开则无法使用

​ 5.对于数组和对象的元素的修改，不算做对常量的修改，不会报错
