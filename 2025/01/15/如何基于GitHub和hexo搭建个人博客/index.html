如何基于GitHub和hexo搭建个人博客

如何基于github搭建自己的个人博客：

​ 本人也是新手小白一枚，看到了很厉害的保姆级博客，然后在各方面的帮助下，成功搭建了属于自己的博客，收获满满，决心把自己踩过的坑和遇到的问题分享出来，可以供大家进行参考。本篇文章也是基于以下教程，参考并进行相应的补充和拓展。

1.前置准备

1.首先要安装好Git ,下面是官方网址,大家可以自行去安装

安装完毕，鼠标右键，看到 Git Bash Here 就显示已经安装好了Git ，可以使用相关命令进行操作

关于Git的常见命令，可以参考下面的文章

然后，记得去github官方网站注册自己的个人账号

2.安装node.js

安装好后，可以对安装成功进行检验

输入 node -v 命令

3.安装Hexo

安装命令：npm install hexo -g

注意：这里有可能出现操作失败的可能，类似下面这样

这是因为权限问题，可以尝试以下方法：

（1）把鼠标放在开始上，鼠标右键，选择终端管理员进行操作

（2）如果还是无法下载，那就再试试下面的办法

尝试以下步骤：
更改执行策略：在PowerShell中输入以下命令以更改执行策略：

Set-ExecutionPolicy RemoteSigned

这将允许远程签名的脚本运行。
重新运行命令：再次尝试运行你的npm命令：

npm install hexo -g

![屏幕截图 2025-01-13 234632](C:\Users\15099\Pictures\Screenshots\屏幕截图 2025-01-13 234632.png)

当时我的采取了第二步以后，就可以下载了，最后输入hexo -v的命令，可以看到安装成功

4.安装hexo依赖

命令：npm install --save hexo-deployer-git
